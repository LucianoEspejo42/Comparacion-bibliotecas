Plotly

¿Qué es?

Una librería de gráficos interactivos, moderna y potente.

Usa JavaScript (Plotly.js) por detrás, pero tiene API en Python.

¿Cómo funciona?

En Python generás un objeto Figure con trazas (Trace) como Scatter, Bar, etc.

El objeto se serializa en JSON.

Luego, en el navegador se usa Plotly.js para renderizar ese JSON.

Fortalezas

  • Interactivo: permite zoom, tooltips, selección, etc.
  • Fácil de integrar en dashboards web (Dash, Flask, Jupyter).
  • Gráficos 3D, mapas, animaciones.

Limitaciones

  • Necesita que el navegador tenga JS habilitado.
  • Más pesado (en cuanto a recursos y tamaño de librerías).

¿Con qué tipo de gráficos puedo trabajar?

Plotly es interactiva y muy rica visualmente. Ideal para dashboards y análisis exploratorio.

  • Líneas (go.Scatter)
  • Dispersión (go.Scatter)
  • Barras (go.Bar)
  • Pastel (go.Pie)
  • Histogramas (go.Histogram)
  • Boxplot (go.Box)
  • Violin (go.Violin)
  • Mapas de calor (go.Heatmap)
  • Subplots y gráficos combinados
  • Gráficos 3D (go.Scatter3d, go.Surface)
  • Mapas geográficos (go.Choropleth, go.Scattergeo)
  • Gráficos de red (usando scatter con mode='lines+markers')
