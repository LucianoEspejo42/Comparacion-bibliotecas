Matplotlib

¿Qué es?

Es la librería de visualización más antigua y estable de Python.

Se usa para gráficos estáticos: líneas, barras, tortas, etc.

Inspirada en MATLAB.

¿Cómo funciona?

Dibuja gráficos usando objetos tipo figura (Figure) y ejes (Axes).

Por defecto, muestra gráficos en una ventana (si hay GUI), pero también se pueden guardar como imágenes (.png, .svg, etc.).

Fortalezas

  • Ideal para informes o PDFs (gráficos no interactivos).
  • Compatible con LaTeX para ecuaciones matemáticas.

Limitaciones

  • No es interactiva (sin zoom ni hover).
  • Menos atractiva visualmente comparada con otras opciones modernas.

¿Con qué tipos de gráfico puedo trabajar?

Matplotlib es la biblioteca más básica y de más bajo nivel, ideal para personalización completa.

  • Líneas (plot)
  • Barras (bar, barh)
  • Histogramas (hist)
  • Dispersión (scatter)
  • Pastel (pie)
  • Boxplot (boxplot)
  • Gráficos de áreas (stackplot)
  • Imágenes (imshow)
  • Contornos (contour)
  • Gráficos 3D (con mpl_toolkits.mplot3d)
  • Subplots personalizados
