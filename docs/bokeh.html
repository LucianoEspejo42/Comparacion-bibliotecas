Bokeh

¿Qué es?

Librería de gráficos interactivos pensada para la web.

Ideal para integrar visualizaciones en dashboards HTML.

¿Cómo funciona?

En Python se define una figura figure(), se agregan glyphs como line, circle, etc.

Bokeh genera automáticamente HTML + JS.

Se inserta en páginas HTML con <div> y <script>.

Fortalezas

  • Muy buena integración con Flask, Django, etc.
  • Altamente personalizable.
  • Interactividad en tiempo real si se conecta con bokeh server.

Limitaciones

  • La sintaxis puede ser más verbosa.
  • No tiene tantos tipos de gráficos como Plotly.

¿Con qué tipo de gráficos puedo trabajar?

Bokeh es interactiva como Plotly, pero está más orientada a visualización web y streaming de datos.

  • Líneas (line)
  • Dispersión (circle, scatter)
  • Barras (vbar, hbar)
  • Histogramas (combinando quad)
  • Boxplot (con Whisker, BoxAnnotation)
  • Gráficos de áreas
  • Gráficos con sliders y widgets
  • Gráficos con zoom, pan, hover
  • Subplots y layouts
  • Mapas (con tile_sources o integración con GeoJSONDataSource)
  • Streaming plots (para datos en tiempo real)
